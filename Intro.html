Visualizating Local Area Unemployment Statistics

An interactive open source software tool for rapid analysis of LAUS estimating local areas' employment and unemployment flow.

This tool will help you gain some knowledge regarding an area's economy. The following data is taken from a public government website. The dataset contains monthly estimates reported of total employmenet and unemployment for approximately 7,300 areas in California. For full details about the derived dataset, visit the Local Area Unemployment Statistics (LAUS) website. Official Government Website: data.edd.ca.gov/Labor-Force-and-Unemployment-Rates

 THIS TOOL IS INTENDED FOR:

  • Look up the selected area's unemployment flow.
  • Download PNG of selected area's graph visualizing the unemployment behaivior. 
  • Stats 128 Professor and Student Project 

 HOW TO USE TOOL:

  • 1. Select Area
  • 2. Select Start Date 
  • 3. Select End State
  • 4. The graph will automatically render 
  • 5. Save and Download graph
